# Assumptions for MVP

- Currency: EGP (editable later).
- Taxes: VAT fields exist but can be zero.
- Single company, single branch (can extend later).
- Arabic UI (RTL) with English identifiers in DB/API.
- Closed accounting system with automatic journal posting from modules.
- Inventory valuation: moving average (can be swapped later).
- Messages: we store outbound logs and hook placeholders for SMS/WhatsApp.
